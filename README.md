# A Profile REST API 

A simple Profile api using Django Rest Framework  with sign-up, login ,users feed (store and edit ), authentication features.

## Tools, Technologies and Libs

Python,
Django 2.2,
Django-Restframework,
SQLite3

## Deployment

Amazon Web Service (EC2 Instance)
